A powerful and easy to use plugin for displaying the total users 

== Description ==

Want to display the total number of your site users? Here it comes, the very powerful and easy to use plugin. 
You can display the total users in seconds. This plugin comes with widget supports many instances, 
and custom PHP function with lots of arguments for your templates. It also comes with the short inline help or instruction for each option, 
so you can easily understand and manage the option only in a single click.

<h4>Key Features & Options</h4>
<ul>
<li><strong>Beautiful user interface</strong> gives you the best experieces while customizing this plugin</li>
<li><strong>No setting page</strong> that will explode your admin page.</li>
<li><strong>Visibility</strong> You can hide a couple roles and the total.</li>
<li><strong>Hide empty</strong> This option will hide the role with empty users</li>
<li><strong>Multiinstances widget</strong></li>
<li><strong>Intro Text</strong> This option will display addtional text before the widget content and supports HTML.</li>
<li><strong>Outro Text</strong> This option will display addtional text after the widget content and supports HTML.</li>
</ul>

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
1. Go to the menu 'Plugins' -> 'Install' and search for 'Total Users Lite'
2. Click 'install'.
